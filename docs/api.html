01 ConANN additions

The adaptive methods.

These methods extend supported IVF indexes beyond a plain fixed-probe FAISS-style CPU workflow.

calibration

calibrate_conann(alpha, k, xq, ground_truth, calib_sz=0.5, tune_sz=0.5, max_distance=100000.0, dataset_key='python')

Calibrate adaptive IVF search from query vectors and exact ground truth. Returns the C++ parameter object together with lamhat, kreg, reg_lambda, and timing fields.

query

search_conann(x, params, k=None)

Run adaptive IVF search and return NumPy distance and label arrays.

evaluation

evaluate_conann(labels, ground_truth, clusters_searched=None)

Compare returned labels with exact ground truth and inspect search quality.

profiling

conann_time_report()

Read timing data collected by the ConANN calibration and search path.
