conann

A Python package for running ConANN-style adaptive IVF search through a familiar FAISS-like API.

Install

Get the package from PyPI

pip install conann

Supported wheels are available for CPython 3.10 through 3.14 on Linux x86_64 and Windows win_amd64. The package is CPU-only.

Quickstart

Run a basic vector search

`conann` accepts contiguous `float32` NumPy arrays, similar to the FAISS Python API. 

import numpy as np
import conann

d = 32
xb = np.random.random((5000, d)).astype("float32")
xq = np.random.random((100, d)).astype("float32")

index = conann.IndexFlatL2(d)
index.add(xb)
distances, labels = index.search(xq, 5)

ConANN API

Calibrate once, search adaptively

This page explains the main workflow. The full generated API reference lists every public import, class, function, constant, and helper exposed by the module.

Open full API reference

calibrate_conann(...)

Computes ConANN calibration parameters from query vectors and exact ground truth.

search_conann(...)

Runs adaptive IVF search using the calibration output.

evaluate_conann(...)

Compares returned labels against ground truth and reports FNR.

conann_time_report()

Returns timing fields from the ConANN calibration/search path.

exact = conann.IndexFlatL2(d)
exact.add(xb)
_, ground_truth = exact.search(xq, k)

quantizer = conann.IndexFlatL2(d)
ivf = conann.IndexIVFFlat(quantizer, d, 64, conann.METRIC_L2)
ivf.train(xb)
ivf.add(xb)

calibration = ivf.calibrate_conann(
    alpha=0.2,
    k=k,
    xq=xq,
    ground_truth=ground_truth,
    calib_sz=0.5,
    tune_sz=0.25,
    dataset_key="example",
)
distances, labels = ivf.search_conann(xq, calibration, k=k)

Build notes

Native wheels are compiled per platform

Linux wheels are built in a manylinux container and repaired with auditwheel. Windows wheels are built with Visual Studio Build Tools, CMake, SWIG, vcpkg, and pyenv-win.

Current scope

  • CPU-only package path
  • ConANN-enabled IVF search
  • Linux x86_64 manylinux wheels
  • Windows win_amd64 wheels
  • No GPU FAISS support in this package
