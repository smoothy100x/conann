02 Quickstart

From vectors to neighbors
in twelve lines.

conann accepts contiguous float32 NumPy arrays, following the core FAISS Python conventions.

quickstart.py
import numpy as np
import conann

d = 32
xb = np.random.random((5000, d)).astype("float32")
xq = np.random.random((100, d)).astype("float32")

index = conann.IndexFlatL2(d)
index.add(xb)
distances, labels = index.search(xq, 5)
A

Shape

Database and query matrices use (n, d) layout.

B

Type

Pass contiguous float32 arrays to the index.

C

Return

search returns distance and label arrays.

03 ConANN workflow

Calibrate once.
Probe per query.

Build exact ground truth for calibration, train an IVF index, then use the calibrated parameters for adaptive search.

input01

Prepare vectors

Use query vectors and exact top-k labels as calibration data.

fit02

Calibrate

Estimate the adaptive stopping parameters for your target false-negative rate.

serve03

Search adaptively

Each query explores IVF lists until its calibrated stopping condition is satisfied.

adaptive_search.py
exact = conann.IndexFlatL2(d)
exact.add(xb)
_, ground_truth = exact.search(xq, k)

quantizer = conann.IndexFlatL2(d)
ivf = conann.IndexIVFFlat(quantizer, d, 64, conann.METRIC_L2)
ivf.train(xb)
ivf.add(xb)

calibration = ivf.calibrate_conann(
    alpha=0.2, k=k, xq=xq, ground_truth=ground_truth,
    calib_sz=0.5, tune_sz=0.25, dataset_key="example",
)
distances, labels = ivf.search_conann(xq, calibration, k=k)

04 Core API

Four methods define
the adaptive path.

calibrate

calibrate_conann(...)

Compute calibration parameters from queries and exact ground truth.

↗ search

search_conann(...)

Run adaptive IVF search with the calibration output.

↗ measure

evaluate_conann(...)

Compare returned labels with ground truth and report FNR.

↗ inspect

conann_time_report()

Read timing fields from calibration and search.
